### git config --list
to view config file

  
### git config --global user.name "YOUR_NAME"
to add username

  
### git config --global user.email "YOUR_EMAIL_ADDRESS"
to add email 


### git clone GIT_URL
to clone a project in


### git remote -v
to view origin


### git push origin master
to push to github.com (must set origin first)
